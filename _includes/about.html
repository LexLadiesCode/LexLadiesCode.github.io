About

LexLadiesCode is an all-ladies coding club based in Lexington, Kentucky. The original ladies came together to create the LexPlayPacman project for the Lexington Public Library. The project was such a success that we decided to continue on, and hope to not only learn more about coding, but teach more women to code and create meaningful and useful tools for the local community.

The club was an idea that sprang from a conversation between Kristen Hanny and Ashley Wilson. Kristen wanted to learn and Ashley had a desire to pass on her knowledge of coding to other women. The club is spearheaded with fellow programming goddess and teacher, Sarah Vessels.
